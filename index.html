JavaScript Practice Projects

A collection of projects built while learning JavaScript.

Calculator App

A simple calculator built with HTML, CSS, and JavaScript that performs basic arithmetic operations.

Counter App

A simple counter built with HTML, CSS, and JavaScript that allows you to increase, decrease, and reset a value with interactive buttons.

Color Flipper

A simple color flipper built with HTML, CSS, and JavaScript that changes the background color randomly and displays the current color code.

Age Calculator

A simple age calculator built with HTML, CSS, and JavaScript that calculates a person's age in years, months, and days based on their date of birth.

Password Strength Indicator

A simple password strength indicator built with HTML, CSS, and JavaScript that evaluates password strength and provides visual feedback based on character requirements.

Image Gallery

A simple image gallery built with HTML, CSS, and JavaScript that allows users to browse images horizontally with smooth scrolling and navigation controls.

Digital Clock

A simple digital clock built with HTML, CSS, and JavaScript that displays the current time in hours, minutes, and seconds with A.M. and P.M. indicators.

Stopwatch

A simple stopwatch built with HTML, CSS, and JavaScript that allows users to start, stop, and reset elapsed time with an interactive interface.
